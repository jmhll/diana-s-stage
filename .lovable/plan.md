

## WebFesDiana — Phase 1: Public Website & Design System

### Design System & Theme
- **Color palette**: Purple primary, mustard yellow accent, elegant neutrals
- **Typography**: Elegant, modern serif/sans-serif pairing suitable for an artistic portfolio
- **Components**: Cards, hero sections, smooth scroll animations, visual blocks with large imagery
- **Mobile-first responsive design** across all pages

### i18n System
- Install and configure a lightweight i18n library (react-i18next)
- Default language: **Catalan**, with Spanish and English
- Language selector in the navigation bar
- Translation files for all three languages with placeholder content

### Navigation
- Responsive navbar with logo, language selector, and menu items
- Mobile hamburger menu with smooth transitions
- Sticky header with transparency effect on scroll
- Footer with social media links, contact info, and quick links

### Pages (all with placeholder content)

**1. Home (Inici)**
- Hero section with large background image/video placeholder and CTA button
- Brief artist introduction
- Quick access cards to portfolio sections
- Highlights of upcoming events

**2. Actriu (Actress)**
- Tabs/subsections: Videobook, Photos, Shows, Artistic CV
- Visual grid layout for media
- Embedded video placeholders for videobook

**3. Presentadora (Presenter)**
- Video showcase section for hosted events
- Featured events highlights with cards

**4. Serveis (Services)**
- Three category cards: Psychology, Career Guidance, Training
- Each with subcategories and descriptions
- Clean layout with icons and descriptions

**5. Galeria (Gallery)**
- Visual portfolio grid with photo/video placeholders
- Category filter buttons (actress, presenter, etc.)
- Lightbox/modal for viewing media

**6. Agenda**
- Calendar-style view of upcoming events
- Event cards showing date, title, location, type
- Placeholder events data

**7. Contacte (Contact)**
- Form with: Name, Surname, Phone, Email, Subject, Description
- WhatsApp direct button
- Corporate email display
- Social media links (Instagram, Facebook, TikTok, LinkedIn)

### Animations & UX
- Smooth scroll between sections
- Fade-in animations on scroll for content blocks
- Hover effects on cards and gallery items
- Page transition effects
